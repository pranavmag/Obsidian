Moore's Law states that integrated circuit resources will double every 18 to 24 months. Developers have to anticipate this when they are working on systems, because the resources at the start of a project may differ from the resources at the end of the project. 

The large collection of servers in giant data centers are known as Warehouse Scale Computers (WSCs), companies like Amazon and Google host these with AWS and Azure. They host cloud servers which people pay for to use to create SaaS apps for PMDs and other things.

To keep a high level of abstraction, high level programming languages like C, C++, Java and others were invented. This makes it much easier for programmers to code because these languages are closer to the English language and algebraic notation rather than cryptic language. For languages like C and C++ a compiler is used to translate the code you write into machine language instructions. Some compilers take it in two steps though by first converting the code to assembly language instructions and then the assembler converts that into machine language instructions. That depends on the compiler you use. 

This chapter also introduced some hardware principles as well. It was said that every component of every computer will be in one of five categories: Input, Output, Memory, Datapath, or Control. Input devices could be devices like keyboards, microphones, mice, these feed the computations that are sent to the output devices which display them. Output devices could be like monitor screen and headphones. There are some devices that are both input and output devices like wireless networks. 

One I/O device is a Liquid Crystal Display also known as an LCD display that most PMDS use to get a thin, low power display. The LCD uses an active matrix that has a transistor at every pixel to determine the control current and make sharper images. In a color active matrix LCD it has 3 transistors at each pixel which represents an RGB mask that determines the color components in the final image. 

The central processing unit, also known as the CPU is the part of the computer which contains the datapath and control and signals I/O devices to activates, tests numbers, adds numbers, and so on.

This chapter also introduced types of memory. There are volatile and nonvolatile memory. Dynamic Random Access Memory also known as DRAM is the main memory that is present while programs and data are running, but it is volatile and loses that data when the power is off. Static Random Access Memory also known as SRAM is also a volatile memory used for cache. Cache memory is very fast memory and it's very tiny which makes it expensive compared to DRAM which is like medium speed and much larger in capacity. Then we have Flash memory which is a nonvolatile memory that is used for storage in phones/tablets and SSDs. It wears down after many writes. Another nonvolatile memory is a Magnetic Disk (Hard Drive) which is much cheaper and slower and used for long term storage.

This chapter also talked about how these integrated circuits are made. It all starts with silicon. Silicon is a semiconductor, meaning that it is a poor conductor of electricity. With a chemical process materials are added to silicon to make tiny areas transform into one of three devices: excellent conductors of electricity, excellent insulators from electricity, or areas that can conduct or insulate depending on the conditions. A silicon crystal ingot is thinly sliced into wafers and those wafers have chemical patterns put on them. Each of those patterned wafers may have some defects so a method was made to cut those wafers into dies (chips).
